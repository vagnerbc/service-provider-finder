# Web Home Contract

## Route

- Public route: `/`
- Authentication: not required
- User-facing language: Portuguese (Brazil)

## Required Visible Content

The home page must display:

- Application name: `Faxinal Conectado`
- A Portuguese (Brazil) headline explaining the service provider discovery
  purpose.
- A Portuguese (Brazil) description that communicates fast and practical access
  to local service providers.
- A Portuguese (Brazil) expectation-setting message that the platform is being
  prepared and that full marketplace actions are not yet available.

## Prohibited Visible Content

The home page must not present active user actions for:

- Account creation
- Login
- Service search
- Provider registration
- Service management

## Responsive Contract

The page must remain readable and visually coherent at these minimum viewport
classes:

- Mobile: 360 px wide
- Tablet: 768 px wide
- Desktop: 1280 px wide

No viewport class may require horizontal scrolling for the main content.

## Accessibility Contract

- The page must have a single primary heading containing or directly associated
  with `Faxinal Conectado`.
- Informational sections must be reachable through semantic document structure.
- User-facing accessible names and metadata must be Portuguese (Brazil).
